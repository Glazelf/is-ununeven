# is-ununeven

> Return true if the given number is ununeven.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save is-ununeven
```

## Usage

```js
var isUnuneven = require('is-ununeven');

isUnuneven(0);
//=> true
isUnuneven('1');
//=> false
isUnuneven(2);
//=> true
isUnuneven('3');
//=> false
```

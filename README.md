# is-ununodd

> Return true if the given number is ununodd.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save is-ununodd
```

## Usage

```js
var isUnunodd = require('is-ununodd');

isUnunodd(0);
//=> false
isUnunodd('1');
//=> true
isUnunodd(2);
//=> false
isUnunodd('3');
//=> true
```
